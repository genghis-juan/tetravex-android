Tetravex Android
========

Tetravex Android is an Android version of the classic puzzle game.
The object of the game is to place all the numbered tiles on the board
so that adjacent tiles have matching edges.

Getting Started
--------
1. Install the (Oracle) Java JDK
2. Install the Android SDK and Android Studio
3. Run the Android SDK Manager to download and install the Android 
support libraries
4. Open the project in Android studio. Make sure the build variant
is set to 'debug'.
